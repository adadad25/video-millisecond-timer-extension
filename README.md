# Video Millisecond Timer (Chrome Extension)

A simple Chrome extension that turns video playback into a precision timing tool.

## Features
- Upload and play video inside browser
- Millisecond timer
- Start / Stop controls
- Lap markers
- Export to CSV

## Installation
1. Download or clone this repo
2. Open chrome://extensions/
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the project folder
